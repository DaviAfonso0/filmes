# Movie Explorer

Aplicação desenvolvida em React para busca e visualização de filmes utilizando uma API externa.

## Funcionalidades

* Listagem de filmes em cartaz
* Exibição de filmes populares
* Busca de filmes por nome
* Página de detalhes com informações completas
* Tratamento de carregamento e erros de requisição
* Interface responsiva para desktop e dispositivos móveis

## Tecnologias Utilizadas

* React
* JavaScript
* CSS
* React Router
* API de Filmes

## Objetivo

Este projeto foi desenvolvido com foco em praticar o consumo de APIs, gerenciamento de estado, componentização, roteamento e desenvolvimento de interfaces modernas utilizando React.
